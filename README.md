# Интерактивный квиз для подготовки к экзамену по матстату
**Репозиторий для отработки защиты итогового задания по курсу "Математические методы и основы статистики"**

## Описание
Этот репозиторий содержит справочные материалы и интерактивный квиз для подготовки к устной защите письменного решения итогового задания. Все материалы основаны на лекциях А.В. Куликова и семинарах курса 2025 года.

## Структура итогового задания
Экзаменационная работа включает **5 задач**:
### Задача 1. Формула полной вероятности
Расчёт вероятности перезаключения контракта с использованием формулы полной вероятности и формулы Байеса.
**Основные темы:**
- События и их разбиения
- Условные вероятности $P(B|A_i)$
- Применение формулы: $P(B) = \sum_{i} P(B|A_i) \cdot P(A_i)$

### Задача 2. Математическое ожидание (купоны)
Определение ожидаемого числа покупок для сбора всех уникальных купонов.
**Основные темы:**
- Геометрическое распределение $Geom(p)$
- Линейность математического ожидания
- Формула: $E[X] = \sum_{i=1}^{n} \frac{n}{n-i+1}$

### Задача 3. Критерий Колмогорова-Смирнова
Симуляция распределения статистики Колмогорова $D_n$ и проверка гипотезы о соответствии выборки $Exp(1)$.
**Основные темы:**
- Статистика Колмогорова: $D_n = \sup_x |F_n(x) - F(x)|$
- Эмпирическая функция распределения
- P-value и проверка гипотез


### Задача 4. VaR Backtesting
Бэктестинг 5-дневного VaR на уровне 5% с применением тестов Купика и Христофферсена.
**Основные темы:**
- Value at Risk (VaR)
- Тест Купика (Proportion of Failures): $LRUC \sim \chi^2(1)$
- Тест Кристофферсена (Independence Test): $LRIND \sim \chi^2(1)$
- Анализ зон Базеля (зелёная/жёлтая/красная)

### Задача 5. Линейная регрессия
Построение линейной регрессии $Y = \beta_0 + \beta_1 X + \varepsilon$ и интерпретация результатов.
**Основные темы:**
- МНК-оценки: $\hat{\beta}_1 = \frac{\text{cov}(X,Y)}{\text{var}(X)}$
- Коэффициент детерминации $R^2$
- Интерпретация коэффициентов регрессии

## Как пользоваться репозиторием
### Для самостоятельной подготовки:
1. **Изучите эталонные решения** (см. раздел "Эталонные работы")
2. **Прочитайте `question_list_opus4.5_v3.md`** — справочник из 45 типичных вопросов на защите
3. **Используйте `task-to-materials-mapping-v3.1.md`** — маппинг задач на теорию из лекций
4. **Для задачи 4 (VaR):** обязательно прочитайте `VaR-Backtesting-Best-Practice_v1.md`

### Режимы работы с AI-ассистентом:
#### MODE 1: Проверка решения
Загрузите своё решение (текст/PDF), получите:
- Сравнение с эталоном
- Указание на ошибки/риски
- Список вопросов, которые может задать преподаватель

#### MODE 2: Симуляция защиты (рекомендуется!)
Тренировка устного экзамена:
1. Кратко изложите суть решения конкретной задачи (2-3 предложения)
2. Отвечайте на последовательные вопросы (без вариантов ответов!)
3. Получайте обратную связь: "Принято/Не принято" со ссылками на лекции

#### MODE 3: Концептуальный квиз
Точечная проверка теории с вариантами ответов (A/B/C/D).
